# 🤖 AI Adaptive Tutor using MCP

## 📌 Overview

This project is an **AI-powered adaptive tutor** that helps users learn more effectively through intelligent content processing and interaction. It uses the concept of **MCP (Model Context Protocol)** to manage structured communication between the AI model and multiple learning tools.

---

## 🧠 What is MCP?

**MCP (Model Context Protocol)** is a framework that allows an AI system to:

* Understand user intent
* Route requests to the correct tool
* Maintain structured context across interactions

Instead of a single monolithic AI response, MCP enables a **tool-based architecture**, where different capabilities are handled by specialized modules.

---

## ⚙️ How MCP Works in This Project

1. User provides input (text, topic, or question)
2. MCP analyzes the intent of the request
3. Based on intent, MCP selects the appropriate tool
4. The selected tool processes the input
5. AI returns a structured and relevant response

This creates a **modular, scalable, and adaptive learning system**.
